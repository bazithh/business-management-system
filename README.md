# 🏢 Business Management System

A full stack business management web application built with PHP, MySQL, Bootstrap 5 and JavaScript.

## 🔗 Developed By
**Abdul Bazith** | Full Stack Web Developer
🌐 Zithex Dev | Kanyakumari, Tamil Nadu, India

## 📸 Features
- 🔐 Secure Login & Session Management
- 📊 Dashboard with Live Business Statistics
- 📦 Product Management (Add, View, Delete)
- 👥 Customer Management (Add, View, Delete)
- 🧾 Invoice Generation with Auto Total Calculation
- 📉 Automatic Stock Reduction on Invoice
- 📱 Fully Responsive Bootstrap 5 Design

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, Bootstrap 5, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Icons:** Font Awesome 6
- **Server:** XAMPP (Apache)

## ⚙️ How to Run This Project
1. Install XAMPP
2. Clone or download this repository
3. Place the folder in `htdocs` directory
4. Open phpMyAdmin and create database `business_db`
5. Import the SQL file from the `database` folder
6. Open browser and go to `http://localhost/business-management-system`
